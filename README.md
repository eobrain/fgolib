# fgolib

A Clojure library that mimics the standard Go Library, to ease porting
of code from Go to [Functional Go][fgo].

## Usage

Include this in the `:dependencies` section of your `project.clj`
file.

    :dependencies [...
                   [org.eamonn.funcgo/fgolib "0.2.0"]
                   ...]
